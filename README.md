# `@THETIMEINC/prettier-config`

> [Sharing configurations](https://prettier.io/docs/en/configuration.html#sharing-configurations).

## Usage

**Install**:

```bash
$ yarn add --dev @thetime/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@thetime/prettier-config"
}
```
